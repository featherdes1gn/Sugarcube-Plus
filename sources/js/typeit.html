TypeIt - SugarCubeJS+

TypeIt is a plugin that allows users to configure lifelike simulated typing. Its primary function is to automatically type on screen, and is extremely easy to setup and use, despite how many options there are to play around with.


Installing TypeIt is as simple as grabbing the CDN link (<> <>
JS CDN: {{{https://unpkg.com/typeit@8.2.0/dist/index.umd.js}}}
<> <>), and importing it into your project using the standard {{{script src=""}}} function, or directly through Twine's {{{Javascript}}} manager.
