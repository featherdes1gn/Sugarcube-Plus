Toastr - SugarCubeJS+

Toastr is a small javascript plugin that allows the user to display aesthetically pleasing notifications (or, toasts). It is easily configurable through easy-to-understand options listed in the documentation.
The toasts that we use in this demo have been slightly modified to better suit the theme of SugarCube+.
